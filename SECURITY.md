# Güvenlik Politikası

Güvenlik açığı, veri sızıntısı veya kötüye kullanım riski fark ederseniz bunu herkese açık issue olarak paylaşmayın.

## Bildirim İçeriği

Lütfen şunları ekleyin:

- Sorunun kısa açıklaması
- Etkilenen ekran veya özellik
- Tekrar oluşturma adımları
- Varsa ekran görüntüsü veya log

## Kapsam

Şunlar güvenlik kapsamında değerlendirilir:

- Kullanıcı verisinin izinsiz açığa çıkması
- Bildirim veya dosya paylaşımıyla ilgili yetki sorunları
- Uygulama paketinin kötüye kullanılmasına yol açabilecek hatalar

Şunlar güvenlik açığı sayılmaz:

- Genel ürün önerisi
- Yazım hatası
- Tasarım önerisi
